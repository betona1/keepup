## 자세한 설명 (Full description, 최대 4000자)
```
습관은 '기록(Log)'으로 남을 때 진짜가 됩니다.

Log Challenge는 좋은 습관을 매일 인증하고 도장으로 남기는 습관 챌린지 앱입니다.
미루면 사라지는 결심을, 마감 전 알림이 놓치지 않게 붙잡아 줍니다.
혼자서도 완결되는 자기만족용 — 회원가입도, 서버도 없습니다. 모든 기록은 내 폰 안에.

■ 스마트폰으로 하는 다양한 인증
· 사진 인증 — 실행 장면을 촬영하면 날짜·시각 워터마크가 자동으로 찍혀요
· 타이머 인증 — 앱 안에서 목표 시간을 재요 (독서·명상·공부). 명상 모드는 호흡 가이드와 내 음악·유튜브까지
· 녹음 인증 — 발음 연습·낭독을 음성으로 남겨요
· 동영상 인증 — 운동·명상 장면을 짧은 영상으로
· URL 인증 — 스토어 등록·업로드·공유 링크 등 웹주소로

■ 놓치지 않게 붙잡는 알림
· 마감 3시간 / 1시간 / 30분 전 알람 (인증하면 자동으로 꺼져요)
· 미룰수록 알람이 늘어납니다 — 나쁜 버릇을 고치는 압박 설계
· 결과형 목표는 마감 3일 전부터 매일 아침 리마인더

■ 두 가지 챌린지 방식
· 적립형 — 매일(또는 주6일) 실행하며 쌓기
· 결과형 — 목표를 선언하고 주 1회 / 15일 / 1회성으로 결과 인증 (한 주기에 여러 번도 가능)
· 인증 시간대 제한(일찍 일어나기), 소감 필수 옵션까지

■ 도장으로 가득 찬 기록
· 시즌 달력에 인증한 날마다 도장이 찍혀요
· 달성률 %, 최장 연속, 전체 도장 수를 한눈에
· 인증 사진은 타임라인 앨범 + 폰 갤러리에도 자동 저장(영구 보존)

■ 안전한 데이터
· 기기 간 백업/이전(ZIP), 자동 스냅샷 복구
· 앱을 지워도 인증 사진은 갤러리에 남습니다

기간은 내가 정합니다. 30일, 63일, 100일, 1년, 2년 —
작심삼일이 아니라, 도장으로 남는 하루하루로.

* 완주 기록은 keepup.keywordream.com 성과 게시판에 자랑할 수 있어요(선택).
```
